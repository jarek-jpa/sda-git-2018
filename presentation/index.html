Zadanie - haiku

  1. utwórz repozytorium o nazwie haiku
  2. utwórz plik o nazwie haiku.txt i wypełnij go przykładowym haiku ze strony abc.haiku.pl
  3. dodaj plik do repozytorium i utwórz commit
  4. wyswietl stan repozytorium i sprawdź jaki hash ma utworzony commit
  5. zmodyfikuj dowolnie haiku i zapisz jako kolejny commit
  6. przywróć widok (w working dir) poprzedniej wersji haiku
  7. przywróć widok ostatniej wersji haiku
  8. cofnij (w repozytorium) zmiany dokonane w ostatnim commicie
  9. wyswietl stan repozytorium i upewnij sie że zawiera (co najmniej) 3 commity

Zadanie - Java HelloWorld

  1. w wybranym IDE utwórz Javowy projekt HelloWorld
  2. utwórz repozytorium w katalogu projektu (git init bez nazwy)
  3. w katalogu src/main/java/hello utwórz klasę HelloWorld wypisującą tekst "Hello World" na ekran
  4. uruchom program
  5. dodaj plik klasy (tylko HelloWorld.java) do repozytorium i utwórz commit
  6. zmodyfikuj nieco program, tak by wyświetlał inny tekst
  7. dodaj zmieniony plik do repozytorium
  8. wyświetl status repozytorium

Zadanie - refaktoring branch

  1. wykorzystaj repozytorium z zadania HelloWorld
  2. utwórz nowy branch o nazwie refactoring i przełącz sie na niego
  3. zmień kod, tak aby wywołanie System.out.println.. znalazło się w osobnej metodzie
  4. zapisz zmianę jako nowy commit
  5. przełącz się na branch master
  6. dodaj kolejne wywołanie System.out.println("I am here")
  7. zapisz zmianę jako nowy commit
  8. obejrzyj graf commitów w IntelliJ Version Control: Log
